Components:
-----------
- Arduino Nano
- 9V battery
- Nikko Ford F-150 Rc car remote control
- Genuine Potentiometers
- NRF24L01+PA+LNA 
- LM1117-3.3V
- LM1117-5V
- 1 resistor
